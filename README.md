# Catch Fleet Helper

This repository hosts public alpha release builds of the Catch Fleet SDR Helper.

The helper is a small Node.js command-line tool that reads local SBS1/BaseStation data from dump1090 and sends summarized aircraft catch observations to Catch Fleet.

## Alpha usage

Download `catch-helper.cjs` from the latest release.

Run:

```bash
node catch-helper.cjs --key cfhlp_xxxxx
